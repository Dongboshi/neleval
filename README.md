conll03_nel_eval
================

Python evaluation scripts for AIDA-formatted CoNLL data

Simple installation
===================

This assumes that you have virtualenv installed on your system.

```Shell
git clone https://github.com/benhachey/conll03_nel_eval
cd conll03_nel_eval
virtualenv ve
cd ve
source ve/bin/activate
pip install .
conll03_nel_eval.py
```
